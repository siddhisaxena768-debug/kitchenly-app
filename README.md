# 🍳 Kitchenly-app

🚀 **Kitchenly** is a modern kitchen-inspired web application designed to make cooking simple, organized, and enjoyable.

This project was created using a combination of AI-powered tools and creative platforms to rapidly design and prototype a user-friendly interface.

---

## 🌟 Features

* 📖 Browse and explore recipe-style content
* 🎨 Clean and modern UI design
* ⚡ Fast and responsive interface
* 🧠 AI-assisted design and development

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS
* **Design & AI Tools Used:**

  * Lovable (AI app builder)
  * Leonardo AI (visual assets)
  * Adobe tools (design & refinement)

---

## 📂 Project Structure

```
Kitchenly/
│── index.html
│── style.css
│── script.js
│── assets/
│── README.md
```

---

## 🚀 Getting Started

To run this project locally:

1. Clone the repository

```
git clone https://github.com/siddhi123/kitchenly.git
```

2. Open the project folder

3. Run `index.html` in your browser
